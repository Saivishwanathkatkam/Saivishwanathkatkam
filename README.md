## 📌 Featured Projects

### 🔊 [rtpproxy-python](https://github.com/Saivishwanathkatkam/rtpproxy-python)
RTPProxy server built from scratch in Python for real-time SIP media relaying.
- Implements the RTPProxy control protocol
- Asyncio-based RTP forwarding with dynamic port allocation
- Full OpenSIPS integration as a media proxy
- Focused on low-latency media forwarding and NAT traversal

### 📞 [websoftphone](https://github.com/Saivishwanathkatkam/websoftphone)
Browser-based SIP softphone using React, JsSIP, and FreeSWITCH.
- SIP over WebSocket (WSS) with ICE negotiation
- Real-time audio in the browser
- Full call flow: React UI → JsSIP → FreeSWITCH → remote endpoint
- Documents SIP INVITE, SDP offer/answer, and media path setup

### 🧠 [Memory-Allocator](https://github.com/Saivishwanathkatkam/Memory-Allocator)
Custom memory allocator implemented in C.
- Demonstrates dynamic memory management and fragmentation handling
- Low-level systems programming with manual heap control
